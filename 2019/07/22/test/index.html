2.根据要求写程序（共40分）

  1. 用最简单的方法实现删除一个list里面的重复元素 （5分）

  2. 用循环判断字符串是否是回文，例如，’abccba’, ‘abcdcba’, ‘123321’, ‘1234321’这种字符前后对称的字符串就是回文 （5分）

  3. 用Python的lambda函数实现求两个数的和 （5分）

  4. 代码实现对列表 a=[1,2,3,4]中偶数位置的元素进行加3后求和（5分）

  5. 两个升序整型列表A和B，元素个数分别为m和n,将数组B合并到数组A中，合并后数组A保持升序（5分）

  6. 写代码实现计算一个字符串中所有数字子串的和， (5分)

    例如：str1 = ‘heoo23ssd-4and23.45sj10’, 和为: 23-4+23.45+10 = 52.5

  7. 对字典aaa = {‘a’: 10, ‘b’:34, ‘c’:11, ‘d’:22}进行以下操作，请写出代码（10分）

    a.对调key和value,生成一个新字典

    b.输出( (‘a’, 10), (‘b’, 34), (‘c’, 11), (‘d’, 22) )

3.简答题（共30分）

  1. Python中list、tuple有什么区别，主要应用在什么样的场景（6分）
  2. Python是怎样管理内存的（6分）
  3. 函数的调用过程（6分）
  4. 对象方法、静态方法和类方法的区别（6分）
  5. 浅拷贝和深拷贝的区别（6分）
  6. 怎么捕获异常（6分）
