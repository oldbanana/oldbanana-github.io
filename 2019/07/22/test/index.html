2.根据要求写程序（共40分）

  1. 用最简单的方法实现删除一个list里面的重复元素 （5分）

  2. 用循环判断字符串是否是回文，例如，’abccba’, ‘abcdcba’, ‘123321’, ‘1234321’这种字符前后对称的字符串就是回文 （5分）

  3. 用Python的lambda函数实现求两个数的和 （5分）

  4. 代码实现对列表 a=[1,2,3,4]中偶数位置的元素进行加3后求和（5分）
